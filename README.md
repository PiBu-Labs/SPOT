# SPOT

**SPOT** (Spatial Positioning and Orientation Tool) is the iOS companion app for TeAPOT (Terrain-Aligned Positioning and Orientation Toolkit). SPOT provides GNSS position and true heading, displays an AprilTag for phone-headset spatial alignment, and streams timestamped sensor measurements to a mixed reality headset running TeAPOT application for outdoor geospatial registration.
